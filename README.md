📌 Sobre o Projeto

O Yuri-Bot-V3.5 é um bot criado para facilitar tarefas no WhatsApp, oferecendo diversos comandos úteis como downloads, ferramentas, diversão, administração de grupos e muito mais.

Projeto desenvolvido para aprendizado e uso pessoal.

---

⚙️ Tecnologias Utilizadas

- Node.js
- JavaScript
- Baileys (WhatsApp Web API)
- FFmpeg
- Git

---

📦 Instalação

Clone o repositório:

git clone https://github.com/notkk067/Yuri-Bot-V3.5

Entre na pasta:

cd Yuri-Bot-V3.5

Instale as dependências:

npm install

Inicie o bot:

node index.js

---

🔑 Conectar o Bot

1. Execute o bot.
2. Escaneie o QR Code que aparecer no terminal.
3. Aguarde a conexão com o WhatsApp.

📜 Comandos

⚙️ Utilidades

Comando| Descrição
".ping"| Verifica se o bot está online
".menu"| Mostra todos os comandos
".gitclone"| Baixa repositórios do GitHub em ZIP

🎵 Downloads

Comando| Descrição
".play"| Baixa música do YouTube
".ytmp3"| Baixa áudio do YouTube
".ytmp4"| Baixa vídeo do YouTube

👥 Grupos

Comando| Descrição
".kick"| Remove membro
".add"| Adiciona membro
".promote"| Promove a administrador
".demote"| Remove admin
